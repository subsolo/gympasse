Projeto referente ao 1º Bimestre/2018 para a Disciplina de Programação IV.

Descrição:

Sistema Web semelhante ao site do gympass, onde terá as seguintes funcionalidades:

- Cadastro de Academias;
- Lista das Academias;
- Lista de Planos;

Tecnologias utilizadas:

- JSP
- Hibernate
